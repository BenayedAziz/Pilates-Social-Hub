--- HOOKS ---
================================================================================

useTheme():
  - Reads/writes "pilates-theme" in localStorage
  - v2 migration: resets old dark preference to light on first load
  - Returns: { theme, setTheme, toggleTheme }
  - Adds/removes "dark" class on document root

use-api.ts (TanStack Query hooks, all hitting /api/* endpoints):
  - useStudios(q?, neighborhood?): GET /api/studios with optional query params
  - useStudio(id): GET /api/studios/:id
  - useClasses(studioId?, type?): GET /api/classes with optional params
  - useFeed(): GET /api/feed
  - useCreatePost(): POST /api/feed (mutation)
  - useToggleLike(): POST /api/feed/:id/like (mutation)
  - useForum(category?): GET /api/forum with optional category
  - useCreateForumPost(): POST /api/forum (mutation)
  - useVoteForumPost(): POST /api/forum/:id/vote (mutation)
  - useProducts(category?): GET /api/products with optional category
  - useBookings(): GET /api/bookings
  - useCreateBooking(): POST /api/bookings (mutation)
  - useCurrentUser(): GET /api/auth/me (no retry)
  - Query client config: staleTime 30s, retry 1

NOTE: API hooks are defined but pages fall back to mock data (FALLBACK_*) when
API calls fail/return undefined, making the app fully functional with mock data alone.
